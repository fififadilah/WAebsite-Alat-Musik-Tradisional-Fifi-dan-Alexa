# WAebsite-Alat-Musik-Tradisional-Fifi-dan-Alexa
Website edukasi alat musik tradisional untuk anak, dibuat oleh Fifi Fadilah dan Alexa Khairunnisa. Menyediakan materi singkat, foto, video, dan suara alat musik dari berbagai daerah. Dilengkapi kuis interaktif agar anak belajar sambil bermain dan mengenal budaya Indonesia dengan cara yang mudah dan menyenangkan.
